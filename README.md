# Data Jobs Dashboard with Power BI

![Dashboard Page 1](Images\Project1_Image1.png)




## Overview

This Power BI Dashboard analyzes the 2024 data job market using real-world job posting data, including job titles, salaries, and locations. It is designed for **job seekers and career switchers** who want a clear, centralized view of market demand, compensation trends and geographic distribution of data roles.

The report consolidates fragmented labor market information into a single, interactive dashboard that enables users to quickly explore and compare insights across roles.

## Skills & Features

This project demonstrates practical Power BI capabilities:

-   **Data Transformation (ETL)** with **Power Query:** Data cleaning and preparing the raw data for analysis by handling blanks, changing data types and creating new columns
-   **Measures and KPIs:** Key metrics such as `Median Yearly Salary` and `Job Count`
-   **Data Visualization:** `Column, bar, line and area charts` for trend and comparison analysis
-   **Geospatial Analysis:** Map visuals to display global job distribution
-   **KPI Cards & Tables:** High-level indicators and detailed, sortable views
-   **Dashboard Design:** Clean, intuitive layout focused on usability and storytelling
-   **Interactive Reporting:**
    -   **Slicers:** To dynamically filter the report by Job Title
    -   **Buttons & Bookmarks:** for navigation
    -   **Drill-Through:** To navigate from a high-level summary to detailed view
